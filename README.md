# Matchmaking App

A C# desktop application built with WinUI that connects users based on preferences, compatibility, and shared interests.

The application includes profile creation, discovery, matching logic, notifications, and additional features such as personality analysis and a bidding system for increased visibility.

## Main Features

* Profile creation with preferences (age, distance, gender)
* Photo upload and profile customization
* Discover section with pass / like / super-like interactions
* Match system with notifications
* Compatibility-based profile ranking
* Profile editing and management
* Personality questionnaire
* Notifications system
* “Hot Seat” auction feature
* Hidden “Spouse Checker” feature with admin review

A detailed description of the features can be found [here](https://github.com/BalaciSofia/UBB-SE-2026-digital-cupids/blob/main/planning/REQUIREMENTS.pdf)


## Tech Stack

* C#
* WinUI
* MVVM architecture
* Mock data utilities (user & community)
